# 1285q round84 shift-walk + selected-body suffix hosting

Builds on `c5fada1` (`1309q x 1,504,387T`) and moves the route to the 1285q tier.

## Changes

- Replaced the ROUND84 low-q shift-by-22 spill/restore window with a value-identical walk:
  `22x mod-p double -> subtract -> 22x mod-p halve`, selected by
  `ROUND84_SHIFT22_WALK_DOUBLE=1`.
- Added `DIALOG_GCD_SELECTED_BODY_GATE_SUFFIX_CARRIES=23`: the selected no-c_in
  GCD body keeps the low carry prefix on measured borrowed carries, while the
  high suffix self-hosts with ordinary MAJ/UMA. The composite scratch planner is
  adjusted to request only the shortened borrowed prefix.
- Spent one GCD branch comparator bit back (`COMPARE_BITS 49 -> 50`) to land a
  clean Fiat-Shamir island for the new op stream.
- Re-hunted the fixed identity-tail nonce: `DIALOG_TAIL_NONCE=73612`.

- phase-garbage batches: 0
- ancilla-garbage batches: 0
