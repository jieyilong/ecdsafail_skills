The dialog-GCD sidecar packs each group of branch bits with a fixed 6→5 reversible
"round763" compressor. Its input slot is `(b0, b0_and_b1)` where
`b0_and_b1 = b0 AND (v < u)` — so the slot state **(0,1) is structurally
UNREACHABLE** on every GCD input (b0_and_b1=1 implies b0=1). Neither bit is mutated
between its definition and the compressor, so the invariant holds at every
compressor site (forward tobitvector, reverse decompress, apply decompress/clear,
per-step swapper).

compressor's dedup'd CCX gates collapse to CX:

existing all-input `ccx(1,3→4)` bracket dedup, the compressor drops from **9 CCX to
4 CCX per direction**. Forward/inverse remain mutual inverses and a 27-distinct
