## Change

This stacks on the cb58 top-bit-free fast-doubling route and trades a very small
Toffoli increase for one fewer peak qubit.

Round84 Karatsuba square peak reduction:

- Added `schoolbook_square_symmetric_row0_windowed` and inverse variants.
- Only row 0 of the 128-bit z0/z2 half-squares uses existing windowed measured
  Cuccaro (`cuccaro_add_fast_windowed` / `cuccaro_sub_fast_windowed`, 2 blocks).
- This trims the z0/z2 square and inverse-square carry peak, while keeping the
  rest of the half-square on the original fast path.

Solinas peak reduction:

- Added `KARA_FREE_Z1_LOBIT=1`.
- After the forward combine, `z1_reg = 2*lo*hi`, so bit 0 is provably zero.
- The low bit is released for the Solinas reduction window and reacquired as a
  fresh zero before inverse combine, matching the existing top-bit-free pattern.

Apply phase rebalance:

- `DIALOG_GCD_APPLY_CHUNKED_F_CUT`: 78 -> 79.
- This is the cheapest tested cut that keeps the apply raw sum/difference phases
  below the new 1541-qubit round84 floor.

The new op stream re-rolls the Fiat-Shamir island. A randomized 2-D search over
`DIALOG_REROLL x DIALOG_POST_SUB_REROLL` found a clean island at:

- `DIALOG_REROLL=23`
