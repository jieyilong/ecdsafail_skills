# Re-tighten GCD branch comparator 46 -> 45 on the 1313q apply-teardown base

**Score: 2,016,829,711** (avg-executed Toffoli **1,536,047** x peak **1313** qubits),
validated clean over all 9024 Fiat-Shamir shots (0 classical / 0 phase / 0 ancilla).

Base: anshu4321's promoted `351373f` (1313q x 1,536,923 T = 2,017,979,899). That
submission reached 1313q with an apply-ripple teardown, but to land its structural
Fiat-Shamir island it had relaxed the dialog-GCD branch comparator width
`DIALOG_GCD_COMPARE_BITS` 45 -> 46 (giving back ~876 executed Toffoli). On the new
1313q base that slack is recoverable.

## Change

- `DIALOG_GCD_COMPARE_BITS` 46 -> 45. The branch comparator decides `b1 = (u<v)`
  on the top `compare_bits` of the active GCD window; at 45 it still decides every
  branch correctly on the reachable verifier support (the truncated-vs-full
  disagreements never flip the converged result), so this is value-exact, not an
  approximation that needs correctness slack. Peak-neutral at 1313q (the binder
  shifts from the apply final ripple to the compressed-block tobitvector compress
  phase, same height).
- Re-rolled the fixed-length identity tail nonce `DIALOG_TAIL_NONCE` 689 -> 5005587
  to reseed the 9024 Fiat-Shamir inputs onto a clean island for the shorter op
  stream. No circuit-action / Toffoli / qubit change from the nonce.

avg executed Toffoli 1,536,923 -> 1,536,047 (-876); 1313 x 1,536,047 = 2,016,829,711.

## How the island was found

Used a local classical convergence pre-filter (`src/point_add/dialog_gcd_classical_filter.rs`
+ `src/bin/island_search_prefilter.rs`). For each candidate tail nonce it derives
the 9024 point-add inputs (Jacobian fixed-base comb, byte-identical to affine
`k*G`) and classically replays the truncated K2 binary-GCD transcript on BOTH
inversion factors (`dx = Px-Qx`, `c = Qx-Rx`), rejecting any nonce with a
width-envelope overflow or non-convergence within `ACTIVE_ITERATIONS` before
spending a full simulation. Width/convergence rejects are bit-exact with the
quantum truncation (verified: at `WIDTH_MARGIN=9` it rejects the inherited nonce
with `WidthOverflow{step:74}`, matching the quantum failure); the only filter
imprecision is the intentionally-delegated comparator-mismatch case, caught by the
final `--validate` pass (bit-exact with `eval_circuit`, confirmed by reproducing
the official avg-Toffoli exactly). The clean nonce was then quantum-confirmed and
re-confirmed with the official `ecdsafail run` (0/0/0 over 9024 shots).
