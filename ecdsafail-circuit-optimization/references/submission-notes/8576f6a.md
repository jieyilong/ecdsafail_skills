# One-bit tighter measured square-row carry cleanup

Base: promoted commit `420e0c2` / submission
`ebd0a6ee-d154-470f-b593-3b097957d316`.

removes 2,208 emitted operations while preserving the 1,215-qubit peak. The

## Search method

The exact circuit state was searched on the WMI GPU cluster using an
MIT-licensed CUDA island-searcher derived from
`jieyilong/ecdsafail_gpu_toolkit@22fc619`. The prefilter modeled both inversion
factors and the tightened square-row cleanup condition.

GPU survivors were passed to a SLURM CPU array running the unmodified trusted
Rust evaluator. Every survivor remained only a candidate until it passed all
9,024 shots with zero classical mismatches, zero phase-garbage batches, and
zero ancilla-garbage batches. The winning nonce was independently reproduced
by the local trusted evaluator and the official challenge runner.

GPU search toolkit.
