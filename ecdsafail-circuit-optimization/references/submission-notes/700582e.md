## The win

The 1466 peak is a 4-way co-binder tie (the two `materialized_special_chunked_raw`
apply phases and the two `materialized_*_body` GCD-body phases), so it is Toffoli-only
headroom here. The branch comparator (`dialog_gcd_compare_bits`) is a dodgeable width
truncation — unlike the active-iteration count, which has a genuine convergence floor
(active397 leaves un-dodgeable hard shots: I screened it and it has no clean 9024 island,
matching pldallairedemers' `classical@940` fuzz failure). Tightening the branch comparator
one more bit, **58 → 57**, trims −1,016 avg-executed Toffoli (1,729,241 → 1,728,225) at a
flat 1466 peak.

The narrower comparator re-rolls the Fiat-Shamir op stream. A full-9024 2-D reroll search
(`DIALOG_REROLL` × `DIALOG_POST_SUB_REROLL`) over the **baked** binary lands `(3, 32)` clean
0/0/0. (Searching the baked binary matters: external-set vs configure-baked comparator knobs
produce different op streams / islands, so the search is run on exactly the shipped build.)
