## Approach

The promoted W-TRUNC submission narrowed Kaliski width loops using an empirical safe envelope. This change re-applies the previously validated UV STEP1 correction fanout on top of that baseline.

In the UV-merged Kaliski bulk prefix, the deferred `(u, v_w)` frame requires the STEP1 correction:

`t = frame & (u0 xor v0)`

The source still emitted two CCX gates with identical controls to apply that correction to `a_f` and `m_i`. This submission computes the temp AND once, fans it out with CX, then measurement-uncomputes it with `CZ(frame, u0 xor v0)` while the XOR value is still live.
